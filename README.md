# CounterFlow

A minimalist counter application with increment and decrement functionality

## Project Vision

A simple counter app with increment and decrement buttons

## Tech Stack

**Frontend**: Vanilla JavaScript + HTML + CSS
**Backend**: None (Client-side only)
**Deployment**: railway

## Features

1. **Counter Display** - Large, centered numeric display showing the current counter value with smooth transition animations
2. **Increment Button** - Button that increases the counter value by 1 with visual feedback on click
3. **Decrement Button** - Button that decreases the counter value by 1 with visual feedback on click
4. **Reset Button** - Button to reset the counter back to zero with confirmation prompt
5. **localStorage Persistence** - Automatically save counter value to browser localStorage so it persists across page refreshes
6. **Keyboard Shortcuts** - Support arrow up/down keys for increment/decrement and 'R' key for reset
7. **Click Animation** - Visual feedback with button press animations and counter value change transitions
8. **Dark Mode Toggle** - Switch between light and dark color themes with preference saved to localStorage
